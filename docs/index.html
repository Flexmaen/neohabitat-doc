Neohabitat

Neohabitat is a relaunch of the first massively mulitplayer online roleplaying game, Lucasfilm's Habitat. Originally developed in 1985 by Lucasfilm Games (now Lucas Arts) for the Commodore 64, Habitat is considered the grandfather of modern MMORPGs. Neohabitat is an open-source project led by Randy Farmer, one of Habitats creators, reviving the original Habitat world.
